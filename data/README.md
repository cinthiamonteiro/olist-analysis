## Dados

Este projeto utiliza o dataset público da Olist, disponibilizado através do Kaggle, conforme link disponibilizado no README principal.

Por limitações de tamanho do GitHub, os dados brutos e o banco SQLite final (.db) não estão versionados neste repositório.

O fluxo de utilização dos dados ocorreu de acordo com as etapas apresentadas abaixo:

1. Extração dos dados da fonte original

2. Limpeza, padronização e validação via Python

3. Persistência dos dados tratados em um banco SQLite (.db)

4. Consumo do banco no Power BI para criação do dashboard

O detalhamento completo das etapas de tratamento encontra-se em 'docs/tratamento_dados.md'.
