## Dados

Este projeto utiliza o dataset público da Olist, disponibilizado através do Kaggle, conforme link disponibilizado no README principal.

Devido a limitações de tamanho do GitHub, os dados brutos e o banco de dados final em SQLite (.db) não estão versionados neste repositório

Esta pasta representa conceitualmente as etapas de tratamento e persistência dos dados dentro do fluxo analítico do projeto, que seguiu as etapas abaixo:

- **Extração dos dados a partir da fonte original**

- **Limpeza, padronização e validação utilizando Python**

- **Persistência dos dados tratados em um banco SQLite (.db)**

- **Consumo do banco no Power BI para construção do dashboard**

O detalhamento completo das etapas de tratamento encontra-se em _'docs/tratamento_dados.md'_ e o banco de dados pode ser acessado no link a seguir [Repositório Google Drive](https://drive.google.com/drive/folders/18YKWzozjJjoJCzyFJHrzurSAhEMPEArx?usp=sharing).
