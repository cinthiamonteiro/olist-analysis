
# Dashboards

Este dashboard foi desenvolvido no Power BI Desktop com base em um banco SQLite tratado e modelado para análises analíticas.

Por limitações de tamanho do GitHub, o arquivo .pbix não está versionado neste repositório.

## Aba – Vendas
Principais indicadores:
- Receita total
- Evolução temporal da receita
- Ticket médio
- Categorias mais representativas
